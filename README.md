## Usage

Copy and run the Repository

## Installation

### First clone the repository form GitHub:
```shell
git clone https://github.com/JoseLuria/react-weather.git
```

### Then install the dependencies:
```shell
npm install
```

#### Finally, start the server with the following command:
```shell
npm run dev
```
